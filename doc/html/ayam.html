Ayam

Reference Manual

Randolf Schultz (randolf.schultz@gmail.com)

24. Jun 2022
This is the documentation of Ayam 1.31 – a free 3D modelling environment for the RenderMan interface. Please note, that this document is a reference manual, more detailed explanations of how to actually model with Ayam are given in the tutorials. This document has been written using the SGML-Tools (LinuxDoc) formatting system to generate files in a variety of text formats from one source file. There are HTML and PDF versions of this document prepared. In addition, the provided SGML-source can be used to generate other formats.
